FUNCIÓN run(p): // p: struct cargado desde archivo

  [1] Inicializar estado s(p)
        ▸ Posiciones de skyrmions  // (triangular / aleatoria / archivo)
        ▸ Posiciones de sitios de anclaje // (aleatoria / archivo)
        ▸ Celdas de anclaje // (si aplica -> Cell Linked-List Algorithm)

  [2] Calcular fuerzas y velocidades iniciales
        ▸ update_vel() // Fss + Fp  ← kernels CUDA	

  [3] Escribir snapshot inicial

  [4] For (i = 0; i < n_steps; ++i) // Loop temporal
        ▸ Integrar posiciones  step(Euler o RK2)
        ▸ Actualizar velocidades update_vel()
        ▸ Guardar datos cada cierto intervalo

  [5] Snapshot final y cierre de archivos

KERNEL force_ss(posiciones x, y → velocidades vx, vy):
extern __shared__ Real sh[];
  
i ← índice global del hilo

  Para cada tile de partículas j:
    │ Cargar x[j], y[j] en memoria compartida
    │ Sincronizar hilos
    │
    └─ Para cada j en el tile:
         dx, dy ← distancia mínima (i, j)
         r      ← ||(dx, dy)||
         fmag   ← K₁(r/λ)    // Bessel modificada
         fxi   += fmag · dx/r
         fyi   += fmag · dy/r
      Sincronizar hilos

  vx[i] ← v0x + factor·(fxi + κ·fyi)
  vy[i] ← v0y + factor·(fyi − κ·fxi)
  
  ---
  
  KERNEL force_pin(x, y, px, py → acumula en vx, vy):

  i ← índice global del hilo
  Si i ≥ N: salir

  xi, yi ← posición del skyrmion // desde mem. global

  fxi, fyi ← 0

  Para cada sitio de anclaje j = 0 ... N_pins − 1:
  │
  │  dx, dy ← (xi − px[j], yi − py[j])
  │  Aplicar imagen mínima // PBC
  │
  │  r² ← dx² + dy²
  │  Si r² ≥ rp²: continu // fuera del radio de anclaje
  │
  └─ fxi −= (fp/rp) · d // fuerza parabólica hacia el centro
     fyi −= (fp/rp) · dy

  // Acumular contribución :
  vx[i] += factor · (fxi + κ · fyi)  ← término Magnus
  vy[i] += factor · (fyi − κ · fxi)  ← término Magnus
